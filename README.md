# PayrollEase
PayrollEase is a web-based payroll management system developed as a Final Year Project. It streamlines employee management, attendance tracking, payroll processing, report generation, and payslip distribution.

## Features

- Role-Based Authentication (Admin & Employee)
- Employee Management
- Attendance Management
- Payroll Processing
- Leave Management
- Report Generation
- PDF Payslip Generation
- Secure QR Code-Based Payslip Access
- Email Payslip Distribution
- Chatbot Assistance

## Technologies Used

- HTML5
- CSS3
- Bootstrap
- JavaScript
- PHP
- MySQL
- PHPMailer
- TCPDF
- Composer
